Glossaire technique
-------------------

.. xglossary::
    :sorted:

    ApplicationWeb
        Une application web est une application disponible sur le web.


    CSS
        Langage associé au langage `HTML` et permettant de décrire de
        manière séparée la présentation des pages web.

        :reference: https://en.wikipedia.org/wiki/Cascading_Style_Sheets

    HTML
        Langage de balises permettant de définir le contenu du page web.
        Ce langage est typiquement utilisé avec le langage `CSS`. HTML permet
        de définir le contenu, `CSS` l'apparence.

    UseOCL
        Outil de génie logiciel destiné à écrire des spécifications `UML`. Cet outil
        est basé sur le langage `OCL`. Il permet d'écrire des modèles de classes
        ainsi que des modèles d'objets tout en vérifiant la validité des objets
        par rapport aux classes.

        :reference: http://scribestools.readthedocs.org/en/latest/useocl/

    DAO
    DataAccessObject
        Une DAO est une classe java permettant de communiquer avec une base de donnée.

    MVC
    ModeleVueControleur
        Le patron MVC est une méthode de programmation permettant d'organiser le programme proporement.

    Controleur
        Dans le cadre d'un patron `MVC` le `Controleur` redirige les requètes envoyées par le client vers les différentes pages de l'application.

    Modele
        Dans le cadre d'un patron `MVC` le `Modele` est une classe java représentant un concept.

    Vue
        Dans le cadre d'un patron `MVC` la `Vue` est une page jsp dynamique.
